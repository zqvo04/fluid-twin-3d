# FluidTwin 3D — 산업 판매 수준 슈퍼 업그레이드 설계안 (v2.0)

> 목표: 실제 산업 현장의 배관 엔지니어가 **파이프라인 문제 진단·신규 설계 검토에
> 실무 참고 자료로 쓸 수 있고, 산업체에 유상 판매가 가능한** 수준의 제품.
> 상태: **격차 진단 + 업그레이드 설계 — 승인 후 Phase 7부터 착수**

---

## 0. 요약 (Executive Summary)

현재 FluidTwin 3D는 **"물리 엔진의 뼈대는 올바르게 서 있으나, 제품은 아직
데모"** 단계다. GGA 정상해석·MOC 과도해석·B31.3 판정이라는 골격은 상용 도구
(AFT Fathom/Impulse, Bentley OpenFlows HAMMER, Pipe-Flo)와 같은 계열이고
교과서 벤치마크 검증도 되어 있다. 그러나 판매 가능한 제품과의 거리는 물리
코어보다 **(1) 해석 적용 범위, (2) 엔지니어링 워크플로, (3) 제품 인프라**에서
훨씬 크다.

| 축 | 현재 | 판매 최소선 | 격차 |
|---|---|---|---|
| 해석 신뢰성·범위 | 물 전용, 4개 구경, 단일 재질, 정상 마찰만 | 다유체·실무 카탈로그·V&V 문서 | **치명적** |
| 엔지니어링 워크플로 | SI 고정, 단일 시나리오, 데이터 교환 없음 | 단위계 전환, 시나리오 비교, EPANET/CSV 연동, 제출용 리포트 | **크다** |
| 제품 인프라 | 계정·저장·과금·문서 전무 | 클라우드 프로젝트, 라이선스, 온보딩, 법적 고지 | **크다** |
| 3D/UX | 게임식 빌더 (강점이자 약점) | CAD식 정밀 입력 + 지금의 직관성 | 중간 |
| 성능 | 수백 파이프 (dense O(n³)) | 수천 파이프, SAB 스트림 | 중간 |

**전략 제안:** 데스크톱 헤비웨이트(AFT 등, 좌석당 수천$)와 해석 범위로 정면
승부하지 않는다. 차별점은 **브라우저 즉시 실행 + 3D 시각화 + 공유/협업 +
자동 취약점 스캔**이다. "설치 없이 링크 하나로 플랜트 수격 리스크를 3D로
보여주는 스크리닝·커뮤니케이션 도구"로 포지셔닝하고, 해석 신뢰성은 그
포지션이 요구하는 만큼(스크리닝 등급 + 투명한 V&V)을 정직하게 달성한다.

## 2. 업그레이드 설계 — Phase 7–12

우선순위 원칙: **신뢰성 → 워크플로 → 제품화** 순. 엔지니어링 도구는 틀린
결과를 한 번 내는 순간 신뢰를 잃고 끝나므로, 화려한 기능보다 "적용 범위 안에서
정확하고, 범위 밖이면 정직하게 거부"가 먼저다.

### Phase 7 — 해석 신뢰성 & V&V (판매의 최소조건) `[G1,G2,G4–G9]`

**7a. 다유체 물성 라이브러리** — `domain/fluid.ts`를 유체 레지스트리로 확장.
- 내장: 물(현행), 에틸렌글리콜 수용액(농도별), 해수, 경유(diesel), 윤활유
  ISO VG 32/68, 응축수. 각각 ρ, μ, p_v, K 의 온도 함수.
- 사용자 정의 유체: 4개 물성을 상수 또는 온도 테이블로 직접 입력.
- 검증: 각 유체 물성을 문헌값 대비 ±2% 이내 스냅숏 테스트.

**7b. 카탈로그 확장** — `domain/catalog/`.
- 배관: NPS ½–24" (½, ¾, 1, 1½, 2, 3, 4, 6, 8, 10, 12, 16, 20, 24),
  Sch 10/40/80/160 (B36.10M/B36.19M 표 데이터).
- 재질: A106-B(현행), A53-B, A312-TP304L/TP316L, HDPE PE4710 —
  각 재질에 **온도별 허용응력 테이블**(B31.3 Table A-1 발췌)과 E(T), 조도.
- 부품: 체크밸브(스윙/노즐 — 역류 슬램 동특성 파라미터), Y-스트레이너,
  오리피스 플레이트, 릴리프 밸브(설정압+블로다운), 진공 브레이커, 서지탱크
  (개방형), 펌프 3계열(소형 인라인/현행 프로세스/대유량 스플릿케이스).
- 검증: 카탈로그 전 항목 치수·K값 스냅숏 테스트, 리듀서 자동 제안이 신규
  구경 조합에서 동작.

**7c. 과도해석 정밀화**
- Brunone 비정상 마찰항 (k₃ 계수, Vardy-Brown 전단 감쇠 근사) — 파형 감쇠를
  실측 수준으로. `physics/transient.ts`/`networkTransient.ts` 공통.
- Suter 4상한 펌프 특성 (Wylie & Streeter 수록 무차원 곡선 내장) — 트립 후
  역류·역회전 영역까지.
- DVCM → **DGCM** 전환(가중계수 ψ, 최소 가스 체적) — 채터링 억제.
- 검증: ① Simpson & Bergant(1994) 컬럼 분리 실험 파형 대비 피크 시각/크기
  비교 ② Wylie & Streeter 펌프 트립 예제 재현 ③ 기존 Joukowsky/4L/a 스위트
  회귀 유지.

**7d. 수치 코어 강화**
- 희소 솔버: GGA의 Schur 보완 행렬을 CSR로 조립, 희소 Cholesky(또는 Jacobi
  전처리 CG). 목표: **파이프 5,000개 정상해 < 2 s** (현 dense는 수백 한계).
- 수렴 실패 진단기: 미수렴 시 잔차 최대 노드, 고립 서브그래프, 제로 유량
  경로, 비현실 경계(양끝 고정수두 충돌)를 검출해 **해당 요소를 3D에서 하이라이트
  + 수정 제안 문구** 출력.
- 검증: 고의로 병든 네트워크 6종(고립 노드, 폐쇄 루프, 수두 충돌 등)에서
  진단 메시지가 정확한 요소를 지목.

**7e. V&V 자동 문서화** — 판매의 핵심 신뢰 자산.
- 벤치마크 테스트 스위트가 실행 시 **오차표를 데이터로 출력** →
  `docs/VALIDATION.md` 자동 생성(케이스, 이론/문헌값, 계산값, 오차 %).
- 제품 내 "Validation" 탭에서 같은 표를 노출 — "이 도구는 무엇으로 검증됐는가"에
  1클릭 답변. 가능 범위에서 EPANET 동일 망 교차 비교(7f의 INP import 활용)를
  포함.

### Phase 8 — 엔지니어링 워크플로 `[G10–G16]`

**8a. 단위계 시스템** — domain/physics는 SI 유지(불변), **표시 계층에서만 변환**.
- `ui/units` 프리젠테이션 모듈: SI ↔ US customary(psi, gpm, ft, in) 전역 토글
  + 항목별 오버라이드. 모든 수치 입력 필드는 단위 접미사를 파싱("150 psi" 입력
  허용).
- 검증: 왕복 변환 무손실, 전 패널·리포트·3D 라벨이 토글에 반응하는 스냅숏.

**8b. 프로젝트 설계 기준** — 프로젝트 설정 객체(직렬화 포함).
- 설계압력/설계온도, 적용 코드(B31.3 현행 / B31.4 / B31.1 — 판정식 계수 분기),
  부식여유 오버라이드, 기본 유체·온도, 단위계.
- 판정 로직이 카탈로그 고정값 대신 프로젝트 기준을 참조하도록 `analysis/`
  리팩터.

**8c. Undo/Redo** — `domain/edit.ts`가 이미 순수 함수이므로 커맨드 스택
(과거/미래 네트워크 스냅숏, 구조 공유)으로 구현. Ctrl+Z/Y, 히스토리 50단계.
- 검증: 전 편집 연산(배치/연결/속성/삭제/탭인)에 대해 undo→redo 왕복 동등성
  프로퍼티 테스트.

**8d. 시나리오 매니저**
- 프로젝트당 복수 시나리오(경계 이벤트 집합: 밸브 궤적, 펌프 트립 시각, 수요
  변동) 저장·복제·이름 관리.
- **비교 뷰**: 두 시나리오의 압력 포락선/피크표를 겹쳐 표시 — "방호 전 vs 후"가
  한 화면에.
- **파라미터 스윕**: 밸브 폐쇄시간 등 1개 변수를 범위 실행 → 피크압 vs 변수
  곡선 (워커 배치 실행, 진행률 표시). "몇 초보다 천천히 닫으면 안전한가"에
  숫자로 답하는 기능 — 스크리닝 도구의 킬러 유스케이스.

**8e. EPANET INP import/export**
- INP 파서/생성기 (JUNCTIONS/RESERVOIRS/TANKS/PIPES/PUMPS/VALVES/COORDINATES
  서브셋). 좌표 없으면 자동 레이아웃.
- 효과: 수도망 시장의 기존 자산을 그대로 열고, EPANET과의 교차검증(7e)이
  성립.
- 검증: EPANET 예제망(Net1 등) 왕복 후 정상해가 EPANET 결과 대비 ±1%.

**8f. 리포트 v2 — 제출 문서 수준**
- 표지(프로젝트명/번호/리비전/작성·검토자/로고 업로드), 목차, 설계 기준 요약,
  모델 요약(BOM), 판정 근거 **수식과 코드 조항**을 각 결과 옆에 병기, 압력
  포락선·시나리오 비교 도표(SVG), 3D 뷰 캡처 삽입.
- 브라우저 인쇄 CSS 기반 **PDF 출력** (외부 의존 없이 print-to-PDF 최적화).

### Phase 9 — CAD급 편집기 & 대형 모델 탐색 `[G16,G25,G26,G24]`

- **정밀 입력**: 배치 중 좌표/길이/각도 키보드 입력(예: "12.5 →" = 현재 방향
  12.5 m), 직교 스냅(45°/90°), 그리드 간격 설정, 기존 노드 좌표 정밀 편집.
- **모델 브라우저**: 좌측 트리(라인/서브어셈블리/노드/컴포넌트 계층), 텍스트
  검색, 라인 번호 태깅(예: `CW-1201-6"-A1`), 트리 클릭 → 3D 플라이투(기존
  기능 재사용).
- **뷰 도구**: 단면(클리핑 평면), 두 점 측정, 라인별 표시/숨김, 경보만 보기
  필터.
- **성능**: SAB 이중버퍼 전환(헤더는 이미 배포에 준비됨, `worker/protocol.ts`에
  예정 지점 명시됨), 3D 라벨을 거리 기반 컬링/집계(가까운 N개만 HTML, 나머지는
  인스턴스드 스프라이트), 목표 **5,000 파이프 @ 60 fps**.

### Phase 10 — 제품 인프라 `[G17–G22]`

- **계정 & 클라우드 프로젝트**: Supabase(Auth + Postgres + RLS) — 프로젝트
  CRUD, 자동저장(디바운스), 리비전 히스토리(저장 시점 스냅숏, 복원). 로컬
  JSON 내보내기는 유지(오프라인/기밀 현장 대응).
- **공유 링크**: 읽기 전용 뷰어 URL(모델+결과 스냅숏) — 보고서 링크 하나로
  상사/고객이 브라우저에서 3D 결과를 돌려봄. 영업 관점 최우선 기능.
- **라이선스/과금**: Free(파이프 50개, 스크리닝) / Pro(무제한, 리포트 브랜딩,
  스윕) / Team(좌석·공유 프로젝트). Stripe 결제, 기능 플래그로 게이팅.
- **품질**: Playwright E2E 스모크(빌드→해석→리포트 3경로), Sentry 오류 리포팅,
  익명 사용 텔레메트리(옵트인).
- **법적 고지(필수)**: 최초 실행·리포트 하단에 명시 — *"본 도구는 예비
  설계/스크리닝 목적의 해석을 제공하며, 결과는 자격을 갖춘 엔지니어의 독립
  검증 없이 최종 설계·안전 판단에 사용할 수 없다."* + 이용약관/책임 제한.
  (엔지니어링 SW의 표준 관행이자 판매의 전제조건.)
- **온보딩**: 첫 방문 인터랙티브 투어(예제 열기→밸브 닫기→서지 확인→에어챔버
  설치→리포트), 문서 사이트(사용법 + **이론 매뉴얼**: 지배방정식·가정·적용
  한계 명문화 — V&V와 함께 신뢰의 양대 축).

### Phase 11 — 차별화 기능 (경쟁 도구에 없는 것) `[영업 무기]`

- **자동 취약점 스캔**: 모든 밸브 급폐 + 모든 펌프 트립 시나리오를 배치
  실행 → **"이 플랜트에서 가장 위험한 이벤트 Top N"** 순위표(피크압, 위반
  조항, 위치 링크). 클릭 한 번으로 플랜트 전체 수격 리스크 지도를 얻는 기능 —
  데스크톱 도구들은 시나리오를 하나씩 수동 구성해야 함.
- **방호 설비 자동 사이징**: 목표 피크압을 만족하는 최소 에어챔버 체적/릴리프
  용량을 이분 탐색으로 제시 — "찾고→고치는" 루프를 "찾고→**자동으로 고쳐
  주는**" 루프로.
- **리플레이 & 슬로모션**: 워커 링버퍼(최근 30 s)를 1/100 배속 재생 + 타임라인
  스크러빙 — 기존 로드맵 P9의 완성.
- **계측 데이터 연동**: 현장 압력/유량 CSV 임포트 → 모델 결과와 겹쳐 그리기 +
  간이 보정(조도/국부손실 스케일 피팅) — "디지털 트윈"이라는 이름값을 하는
  최소 기능.

### Phase 12 — 시장 진입 마감

- 공개 데모 사이트(예제 갤러리 6종 — 급수 부스터, 냉각탑 순환, 소방 주관,
  장거리 이송관, 고층 리저 시스템, 480파이프 그리드), 시나리오 딥링크.
- 랜딩 페이지 + 가격표 + V&V/이론 매뉴얼 공개(신뢰 마케팅).
- 파일럿 프로그램: 설계사무소/EPC 3곳에 무상 Pro 제공 ↔ 피드백·사례 확보.
